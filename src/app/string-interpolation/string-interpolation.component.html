String Interpolation examples

Things we can do using String Interpolation

  • Simple code expression example: {{4+5}}
  • First property example: {{num1}}
  • First code expression with Property example: {{num1+num2}}
  • Boolean example: {{flag}}
  • String example: {{name}}
  • String example 2: {{name+" "+name2}}
  • Object example: {{person}}
  • Object example 2: {{person.name}}
  • Object example 3: {{person.age}}
  • Object example 4: {{person.address}}
  • Method example 1: {{getName()}}

Things we cannot do using String Interpolation

  • We cannot create a new object in string interpolation
  • We cannot write multiple code expression in string interpolation
